# 🧹 Data Cleaning & Preparation using Microsoft Excel

## 📌 Project Overview

This project demonstrates the process of cleaning and preparing a retail sales dataset using Microsoft Excel. The primary objective was to improve data quality by handling missing values, checking for duplicate records, and ensuring consistent formatting to make the dataset ready for analysis.

---

## 🎯 Objectives

- Handle missing values
- Check for duplicate records
- Standardize date formats
- Validate numeric fields
- Prepare the dataset for analysis

---

## 🛠️ Tools Used

- Microsoft Excel

---

| Attribute | Details |
|-----------|---------|
| Dataset | Retail Sales Dataset |
| Total Records | 1,200 |
| Total Columns | 14 |
| Tool Used | Microsoft Excel |

---

## 🧹 Data Cleaning Steps

- Identified missing values in the dataset.
- Filled missing values in the **CouponCode** column.
- Verified that there were no duplicate **OrderID** values.
- Standardized date formatting.
- Validated numeric fields.
- Ensured the dataset was clean and analysis-ready.
